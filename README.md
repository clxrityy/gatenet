# gatenet

> [!IMPORTANT]
> This is the development branch for the refactor of gatenet.

---

> `gatenet` is a Python-first networking observation toolkit with a thin, portable CLI for inspecting local networks safely and clearly.

## Quickstart

```bash
make install # Install the package in a virtual environment
make gatenet discover # Discover devices on the local network
```

```bash
make install-dev # Install the package with development dependencies
```
